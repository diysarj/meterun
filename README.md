# Meterun - Running Plan

Meterun is a smart running companion app designed to help you generate personalized training plans, track your progress, and visualize your routes. Whether you're training for a 5K or a Marathon, Meterun adapts to your current fitness level and race goals.

## Features

-   **Smart Training Plans**: Generate adaptive training schedules based on your race date, distance goal (5K, 10K, Half, Full), and current fitness level.
-   **Progress Analytics**: Visual analytics to track your weekly completion rates and distance vs. plan.
-   **Route Visualization**: Explore and visualize running routes on an interactive map.
-   **Calendar Sync**: seamless integration with Google Calendar to keep your training schedule organized.
-   **Pace Calculator**: Tools to help you calculate target paces for different race distances.

## Tech Stack

**Frontend**

-   React.js (Vite)
-   Tailwind CSS (Styling)
-   Framer Motion (Animations)
-   Recharts (Analytics/Charts)
-   React Leaflet (Maps)
-   React Router DOM

**Backend**

-   Node.js & Express.js
-   MongoDB & Mongoose
-   Google OAuth for Strava & Google Calendar API
-   JWT Authentication
